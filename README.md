# Placement Cell Portal

A full-stack MERN application designed to streamline campus placement activities through dedicated portals for Students, Companies, Mentors, and Administrators.

## Features

* Secure JWT Authentication
* Student, Company, Mentor, and Admin Portals
* Job Posting and Management
* Job Application Tracking
* Dashboard and Analytics
* Resume Upload and Management
* Placement Monitoring and Reporting

## Technology Stack

### Frontend

* React.js
* CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend folder and configure the required variables:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

## Project Objective

The Placement Cell Portal simplifies campus recruitment by providing a centralized platform for students, companies, mentors, and administrators to manage placement-related activities efficiently.

Developed using the MERN Stack.
